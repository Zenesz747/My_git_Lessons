## Списки
Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой, например вот так:
* элемент 1
* элемент 2
* элемент 3

Чтобы добавить нумерованные списки, необходимо просто пронумеровать пункты, например вот так:
1. Первый пункт
2. Второй пункт
